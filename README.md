# Setup
## Server
1. Install Dependencies:
```shell
cd server/
python3 -m pip install -r requirements.txt
```
2. Put data files in place.

3. Launching Server:
```shell
cd server/
flask run
```
## Front-end
1. Installing Dependencies:
```shell
cd frontend/
npm i
```
2. Launching Front-end:
```shell
cd frontend/
npm run dev
```
Then open `localhost:5173` in browser.
